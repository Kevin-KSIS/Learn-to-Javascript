## to string
- String(object)

`1 + 'b' = '1b'`

## to number

- Number(object)

| Value	  	| Becomes.. |
|-----------|-----------|
| undefined | NaN  		| 
| null  	| 0  		|  
| true false| 1 0  		|
| string  	| 0 or NaN	|

## to boolean
- Values that are intuitively “empty”, like 0, an empty string, null, undefined and NaN become false.
- Other values become true.
